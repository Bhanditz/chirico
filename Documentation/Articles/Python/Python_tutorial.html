I/O Commands in Python

The following command will create the file 'Stuff' and write to this file. Open the file with 'w+' option enables reading. The command f.seek(0,2) moves to the end of the file. 


"This example Creates and writes to a file"
f=open('Stuff','w+')
f.write('one\n')
f.write('two\n')
f.write('three\n')
f.flush()
f.seek(0)   
a=f.readline().strip()  
"At this point a = 'one'"
f.seek(0,2)
f.write('four\n')
f.write('five\n')
f.flush()
f.seek(0)
a=[t.strip() for t in f]
